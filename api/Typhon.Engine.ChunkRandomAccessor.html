Remarks

This class is not thread-safe. When you access a Chunk, you have the possibility to pin it or not. Pinning is mandatory if you will make other chunk access while you use the first one. Accessing another chunk might kick the page of the one you're using, pinning prevent that. But as long as you don't request another chunk, you are safe.
