Span of the Logical Segment's raw data. See Remarks.

Property Value

Span<byte>

Remarks

If the page block is part of a Logical Segment, this will return the span covering its raw data section. BEWARE: the root page of a Logical Segment is 6000 bytes wide, subsequent pages will be 8000 bytes. Unpredictable result will occur if using this property on a non-Logical Segment Page Block.
