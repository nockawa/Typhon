Remarks

The file is opened in exclusive mode, is composed of 8KiB pages. The user requests a given page by calling RequestPageShared(uint) or RequestPageExclusive(uint), the requested disk page is mounted in memory and can be access through its PageAccessor instance.

The settings are given through DatabaseConfiguration:
- DatabaseCacheSize determines the memory allocated to hold pages, must be a multiple of Typhon.Engine.PagedMemoryMappedFile.PageSize.
- WriteCacheSize determine the memory allocated to copy pages that are logically sequential but not arranged this way in memory in order to write them into the file in one write command. Must be a multiple of Typhon.Engine.PagedMemoryMappedFile.PageSize.

A Least Frequent Used cache evicts the pages when it runs out of memory.
