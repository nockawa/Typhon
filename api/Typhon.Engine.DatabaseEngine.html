Create a transaction in order to make Queries and CRUD operation on the database

public Transaction CreateTransaction()

Returns

Transaction

The transaction object

Remarks

Typhon deals with accesses and changes through transaction only, even for query purpose. When the user creates a transaction, "now" (the time when the transaction was created) is used as the reference point, every access will be based on the data that existed up to this point. Every change will be isolated from other transactions until the content is committed.
