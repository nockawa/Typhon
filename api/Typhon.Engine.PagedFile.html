Parameters

ValueTask<PageAccessor>

The accessor

Remarks

This method goes async if the Memory Page is not allocated and there are no free Memory Pages available or if the requested access requires to wait for the transition to be made (one or many threads hold an access not compatible with the one we request). If the File Page is being loading from disk to memory, the read is completely independent of this operation, the PageAccessor will wait for it upon its first content access. Going async on this call should be very rare.
