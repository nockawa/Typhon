RequestPage(int, bool, out PageAccessor, long, CancellationToken)

Request access to a File Page, either Shared or Exclusive.

public bool RequestPage(int filePageIndex, bool exclusive, out PageAccessor result, long timeout = -1, CancellationToken cancellationToken = default)

Parameters

filePageIndex int

The index of the File Page to access.

exclusive bool

true for Exclusive access, false for Shared.

result PageAccessor

The PageAccessor allowing access to the page, valid only if the method returns true.

timeout long

The time (in tick) the method should wait to return successfully.

cancellationToken CancellationToken

An optional cancellation token for the user to cancel the call.

Returns

bool

true if the call succeeded, result will be valid. false if the operation was cancelled or time out result won't be valid and be set to default.

Remarks

This method will enter a wait cycle if:

  • The Memory Page is not allocated and there are no free Memory Pages available
  • The requested access requires to wait for the transition to be made (one or many threads hold an access not compatible with the one we request).

    • If the File Page is being loading from disk to memory, the read is completely independent of this operation, the PageAccessor will wait for it upon its first content access.
